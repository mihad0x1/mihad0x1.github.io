About Me

I'm Mihad Mehrab, an Independent Security Researcher from Bangladesh with a strong passion for Web Application Security and Bug Bounty Hunting. Since September 2024, I have been actively working with international bug bounty platforms, helping organizations identify and responsibly disclose security vulnerabilities before they can be exploited. My primary areas of expertise include Access Control, Business Logic, Authentication, API Security, IDOR, and Web Application Security.

I enjoy solving complex security challenges, continuously learning new technologies, and sharing knowledge through technical write-ups. My long-term goal is to become one of the leading Security Researchers in the industry while contributing to a safer internet.
